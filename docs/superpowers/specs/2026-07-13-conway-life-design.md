# 康威生命游戏（Tkinter）设计说明

## 目标

实现一个零第三方依赖、可直接运行的 Python Tkinter 小程序，用可视化网格演示康威生命游戏的元胞自动机演化。

## 范围

- 默认网格为 60 列 × 40 行，每个单元格 15 像素。
- 存活细胞显示为深色，死亡细胞显示为浅色。
- 鼠标左键点击网格可切换对应细胞的生死状态。
- 提供“开始 / 暂停”“单步演化”“随机生成”“清空”按钮，并显示当前代数。
- 自动演化间隔为 150 毫秒。
- 网格边缘以死亡细胞处理，不采用环绕边界。

## 结构

程序保持为一个 `conway_life.py` 文件：

- `LifeGame` 类负责保存二维细胞状态，并计算每个细胞的八邻域存活数和下一代状态。
- `LifeApp` 类负责 Tkinter 窗口、Canvas 绘制、按钮回调、鼠标事件和定时更新。
- 演化规则：存活细胞有 2 或 3 个活邻居时存活；死亡细胞恰有 3 个活邻居时复活；其他情况死亡或保持死亡。

## 交互和异常处理

- 演化运行时仍可点击方格调整细胞状态。
- 程序关闭时取消后续定时调度，避免窗口销毁后继续回调。
- 不读取外部文件，也不需要网络或安装依赖。

## 验证

- 用 Python 编译检查确认语法正确。
- 用闪烁器（Blinker）验证两代后恢复初始状态。
- 手动启动窗口，验证按钮、鼠标切换、自动演化和关闭窗口均可用。
